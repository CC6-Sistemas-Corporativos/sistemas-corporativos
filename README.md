## Sistema de Gestão de Candidatos
O projeto consiste em um sistema de gestão de candidatos para vagas de emprego, que centraliza todas as etapas do processo seletivo: desde a publicação da vaga, passando pelo cadastro e acompanhamento dos candidatos, triagem de currículos, entrevistas e avaliações, até os processos finais de contratação. O objetivo é otimizar o recrutamento, garantir organização das informações e dar mais agilidade e transparência à seleção.
